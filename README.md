# TelegrmTravelBot



## Run
configure the path to database in application.properties

```java
spring.datasource.url=jdbc:mysql://localhost:3306/YourBD?useSSL=false&characterEncoding=UTF-8
```

install the dependencies


```bash
mvn install
```
run project

```bash
mvn spring-boot:run
```
